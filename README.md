# Fullstack Task 2 — Registration Form
**Internship Project — Cognify Technologies**

This project demonstrates a **registration form** with server-side validation using:

- HTML & EJS
- Node.js & Express
- Server-side rendering
- Form submission handling with **temporary storage (in-memory array)**
- Basic server-side validation for required fields

---

## How to run

1. Clone the repo:
git clone <repo-url>

2. Install dependencies:
npm install

3. Start the server:
node server.js

4. Open in browser:
http://localhost:3000

---

## Features

- Responsive UI with centered registration form
- Server-side rendering using EJS
- Inline validation for empty fields
- Thank you / success page after submission
- Temporary in-memory storage for submitted data
